# Welcome to SRAM 👋

SRAM is a school project, result of a 24-hour hackathon. Theam: Mars! This project won the first place for using Symfony when the framework was not yet known by wilders...

## Usage

```sh
composer install
yarn install
yarn encore dev
symfony server:start
```
